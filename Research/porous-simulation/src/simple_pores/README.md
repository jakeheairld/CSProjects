# Simple Pore Generation Scripts

This folder contains scripts for generating and processing computational meshes used in multi-physics simulations.

## **Contents**
- `generate_rectangular_pores.py`: Python script to generate meshes with rectangular pores.
- `generate_circular_pores.py`: Python script to generate meshes with circular pores.

## **Usage**
### **Python Script**
To generate a simple 2D porous mesh using the Python script in terminal, run:
```bash
generate_rectangular_pores.py
```
The .msh of this mesh should be saved in the directory **datasets/meshes/msh**

The .xdmf of this mesh should be saved in the directory **datasets/meshes/xdmf**
